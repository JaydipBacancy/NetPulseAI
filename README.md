# NetPulse AI

NetPulse AI is a Next.js full-stack telecom demo for 5G network monitoring, alerting, predefined test execution, AI analysis, and lightweight reporting.

## Tech Stack

- Next.js App Router for frontend and backend workflows
- Next.js Server Components, Server Actions, and Route Handlers
- TypeScript
- Tailwind CSS v4
- shadcn/ui foundations
- Vercel
- pnpm via Corepack

`react` and `react-dom` remain framework dependencies required by Next.js. The application architecture should stay Next.js-first and avoid external backend services.

## Getting Started

Install dependencies and start the development server:

```bash
corepack pnpm install
corepack pnpm dev
```

The app runs at `http://localhost:3000`.

## Build

```bash
corepack pnpm build
```

## Product Scope

- Authentication
- Monitoring dashboard
- Nodes view
- Alert center
- Predefined test runner
- AI analysis
- Simple reports
